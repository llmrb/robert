Robert

Robert is an AI assistant for FreeBSD. It answers questions using official man pages, documentation, the local ports tree, and package metadata. It runs entirely in your terminal and ships as a statically linked, ~3MB binary — no dependencies, no Node.js, no Electron, no browser needed.

Ask questions in plain English. Robert searches man pages, the filesystem, ports, and packages, then answers with cited excerpts from official docs — not its training data.

How It Works

Grounded answers

The system prompt forbids using training data. Every claim must cite a man page via blockquote.

Cooperative scheduler

The LLM call runs in a worker task while the event loop keeps the terminal UI responsive.

Roff sanitisation

Raw man output is cleaned before it reaches the model, preventing garbled paths from roff overstrike formatting and underscore-wrapped paths.

Ports tree lookup

Robert can search and read a local FreeBSD ports tree, using PORTSDIR when set and /usr/ports otherwise.

Package database lookup

Robert can search and read package metadata from the local pkg(8) database.
