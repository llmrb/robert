## About

Robert has been using FreeBSD since the early 2000s,
and the first release he installed was 4.4-RELEASE.
He is an expert at reading and searching manual pages,
and sometimes he can read files.

Robert is also Artificial Intelligence packaged as a
native, 2MB FreeBSD binary. It currently requires
DeepSeek because it is almost free to use but _it could_
also work with OpenAI, Anthropic, Google, xAI, zAI,
and self-hosted options like Ollama. It just hasn't
proven to be worth the effort yet.
